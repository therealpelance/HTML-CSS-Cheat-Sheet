HTML Reference Cheat Sheet

Main Document Components

Tag Name Description

<html>

 Main Root The root level element of an HTML document. All other elements must be descendants of this element.

<head>

 Document Head This element contains the machine-readable metadata about the document. Nothing within this element will be displayed by the browser visually.

<link>

 External Resource Link This element is used to establish a link between the current document and an external resource. Most commonly used to link CSS stylesheets.

<meta>

 HTML Metadata Element This element is used to represent metadata not able to be represented by other HTML meta-related elements.

<style>

 HTML Style Element This element contains CSS style information for the document.

<title>

 Browser Title This element contains the title displayed in a browser's title bar or page tab.

<body>

 Document Body This element is the main content body of the HTML document. It contains everything to be visibly displayed by the browser. There can only be one per document.

Sectioning

Tag Name Description

<address>

 Creator Address This element contains HTML that provides contact information for a person, people, or organization.

<article>

 Article Element This element contains a stand-alone composition that makes sense independent of the rest of the document and is intended for distribution or syndication.

<aside>

 Aside Element This element contains content only indirectly related to the main content. Often used for sidebars.

<footer>

 Footer Element This element specifies the footer for the nearest sectioning parent. Can be page footer or footer of another section.

<header>

 Header Element This element contains introductory content to either the page as a whole or the section representing its parent.

<h1>, <h2>, <h3>,
<h4>, <h5>, <h6>

 Headings These elements represent six levels of section headings. <h1> being the largest and <h6> being the smallest.

<hgroup>

 Heading Group This element represents a multi-level heading for a document section. It contains a set of section headings (<h1> - <h6>).

<main>

 Main Content Element This element contains the dominant content of the document body. This content directly relates to or expands on the central topic or functionality of the document.

<nav>

 Navigation Element This element contains code whose purpose is to provide navigational links. Commonly used for menus, tables of contents, and indexes.

<section>

 Section Element This element represents a stand-alone section of the document. <section> should only be used if a more semantic element does not exist for the content it contains.
