HTML Reference Cheat Sheet

Main Document Components

Tag Name Description
<html> Main Root The root level element of an HTML document. All other elements must be descendants of this element.
<head> Document Head This element contains the machine-readable metadata about the document. Nothing within this element will be displayed by the browser visually.
<link> External Resource Link This element is used to establish a link between the current document and an external resource. Most commonly used to link CSS stylesheets.
<meta> HTML Metadata Element This element is used to represent metadata not able to be represented by other HTML meta-related elements.
<style> HTML Style Element This element contains CSS style information for the document.
<title> Browser Title This element contains the title displayed in a browser's title bar or page tab.
<body> Document Body This element is the main content body of the HTML document. It contains everything to be visibly displayed by the browser. There can only be one per document.
